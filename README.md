faq_package
===========
It manages the dashboard questions and answers, and package for display in the block page.

Dashboard
search FAQ
add / edit FAQ

Block Type
FAQ view

Function
I can register the question and answer in the dashboard, change, delete, and view.
You can to reorder the display with seasrch FAQ of the dashboard. (Using. The jquery ui sortable)
It is displayed as it was sorted in the dashboard at block type.


faq_package
===========
質問と回答をダッシュボードで管理し、ページにはブロックで表示するためのパッケージ。

管理画面
　search FAQ
　add/edit FAQ

ブロックタイプ
　FAQ view

機能
管理画面でquestion and answer の 登録、変更、削除、閲覧ができます。
管理画面の seasrch FAQ で 表示を並び替えができます。（jquery ui sortableを使用。）
ブロックタイプにてダッシュボードで並び替えしたとおりに表示されます。
